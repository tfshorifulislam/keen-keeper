📊 KinKeeper
🚀 Overview

KinKeeper is a modern relationship management web app that helps you track, manage, and strengthen your personal connections. It allows users to monitor interactions like calls, texts, and video chats, ensuring no relationship is neglected.

🛠️ Technologies Used
⚛️ React.js
🔀 React Router DOM
🎨 Tailwind CSS + Component Library
📈 Recharts

✨ Key Features
📅 Smart Timeline

Track all your interactions (calls, texts, video chats) in a clean and filterable timeline.

🎯 Interaction Analytics

Visualize your communication habits with dynamic charts using Recharts.

🔍 Advanced Filtering

Filter interactions by type (Call, Text, Video) to quickly find specific activities.
